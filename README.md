# 42 Core Curriculum

This repository documents my journey through the 42 Common Core curriculum.

42 is a project-based software engineering program focused on peer-to-peer learning, problem solving, and building strong foundations in computer science. Through the Common Core, students develop skills in low-level programming, algorithms, data structures, systems programming, and software engineering.

This repository serves as an overview of my progress through the 42 Common Core.
